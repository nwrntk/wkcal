Direct Substitution Property

Theorem

ถ้า f เป็น polynomial หรือ rational function และ c อยู่ใน domain แล้ว:

limx→c f(x) = f(c)

2.4 One-Sided Limits

Left-Hand Limit

limx→c⁻ f(x) = L

x เข้าใกล้ c จากซ้าย (x < c)

Right-Hand Limit

limx→c⁺ f(x) = L

x เข้าใกล้ c จากขวา (x > c)

Theorem — Existence of Limit

limx→c f(x) = L   ก็ต่อเมื่อ   limx→c⁻ f(x) = L และ limx→c⁺ f(x) = L

2.5 Continuity

Definition — Continuous at c

f ต่อเนื่องที่ c ก็ต่อเมื่อ ทั้ง 3 เงื่อนไข เป็นจริง:

  1. f(c) มีค่า (defined)
  2. limx→c f(x) มีค่า (exists)
  3. limx→c f(x) = f(c)

ประเภทของ Discontinuity

คลิกเพื่อดูแต่ละประเภท

Removable Discontinuity

Limit มีค่า แต่ f(c) ไม่มีค่า หรือ f(c) ≠ Limit

แก้ได้ด้วยการ นิยามใหม่ ที่จุด c

Jump Discontinuity

lim⁻ ≠ lim⁺ → กราฟ "กระโดด" ที่จุด c

พบบ่อยใน Piecewise Functions

Intermediate Value Theorem (IVT)

IVT — Intermediate Value Theorem

ถ้า f ต่อเนื่องบน [a, b] และ N อยู่ระหว่าง f(a) กับ f(b) แล้ว มี c ∈ (a,b) ที่ f(c) = N
